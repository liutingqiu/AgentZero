## 需求

为文件上传校验功能编写单元测试，覆盖文件类型校验和文件大小校验。

### 测试内容

**测试 1：允许的文件类型可以通过**

```python
def test_allowed_extension_txt():
    """.txt 文件应该被允许"""
    ext = '.txt'
    ALLOWED_EXTENSIONS = get_allowed_extensions()  # 从模块导入或直接引用常量
    assert ext in ALLOWED_EXTENSIONS
```

**测试 2：禁止的文件类型被拒绝**

```python
def test_blocked_extension_exe():
    """.exe 文件应该被拒绝"""
    ext = '.exe'
    ALLOWED_EXTENSIONS = get_allowed_extensions()
    assert ext not in ALLOWED_EXTENSIONS
```

**测试 3：常见危险扩展名全部被禁止**

```python
def test_dangerous_extensions_blocked():
    """常见危险扩展名全部不在白名单中"""
    dangerous = ['.exe', '.bat', '.cmd', '.com', '.msi', '.scr', '.pif', '.vbs', '.ps1', '.sh']
    ALLOWED_EXTENSIONS = get_allowed_extensions()
    for ext in dangerous:
        assert ext not in ALLOWED_EXTENSIONS, f'{ext} 不应在白名单中'
```

**测试 4：MAX_FILE_SIZE 为 10MB**

```python
def test_max_file_size():
    """MAX_FILE_SIZE 应为 10MB"""
    MAX_FILE_SIZE = get_max_file_size()
    assert MAX_FILE_SIZE == 10 * 1024 * 1024
```

## 完整测试文件结构

```python
"""文件上传校验单元测试"""
import pytest
import sys
import os

sys.path.insert(0, os.path.join(os.path.dirname(__file__), '..'))

# 从 server.py 导入常量
from app.api.server import ALLOWED_EXTENSIONS, MAX_FILE_SIZE


class TestFileUploadValidation:
    """文件上传校验功能测试"""

    def test_txt_allowed(self):
        assert '.txt' in ALLOWED_EXTENSIONS

    def test_png_allowed(self):
        assert '.png' in ALLOWED_EXTENSIONS

    def test_py_allowed(self):
        assert '.py' in ALLOWED_EXTENSIONS

    def test_exe_blocked(self):
        assert '.exe' not in ALLOWED_EXTENSIONS

    def test_bat_blocked(self):
        assert '.bat' not in ALLOWED_EXTENSIONS

    def test_sh_blocked(self):
        assert '.sh' not in ALLOWED_EXTENSIONS

    def test_ps1_blocked(self):
        assert '.ps1' not in ALLOWED_EXTENSIONS

    def test_msi_blocked(self):
        assert '.msi' not in ALLOWED_EXTENSIONS

    def test_dangerous_extensions_all_blocked(self):
        dangerous = ['.exe', '.bat', '.cmd', '.com', '.msi', '.scr', '.pif', '.vbs', '.ps1', '.sh']
        for ext in dangerous:
            assert ext not in ALLOWED_EXTENSIONS, f'{ext} 不应在白名单中'

    def test_max_file_size_is_10mb(self):
        assert MAX_FILE_SIZE == 10 * 1024 * 1024

    def test_allowed_extensions_are_lowercase(self):
        for ext in ALLOWED_EXTENSIONS:
            assert ext == ext.lower(), f'{ext} 应该是小写'
```

## 遇到问题时的决策树

1. `ALLOWED_EXTENSIONS` 或 `MAX_FILE_SIZE` 不在模块级别定义（在函数内部）→ 停止，报告（需要先重构 server.py 将其提升为模块级常量）
2. 导入报错 → 添加 `sys.path.insert` 或设置 `PYTHONPATH`
3. 测试运行失败 → 检查实际值与预期值是否一致
