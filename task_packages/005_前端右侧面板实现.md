### JS 部分（`app.js` 文件末尾追加）

**新增以下函数**：

1. `initRightPanel()` — 创建右侧面板 DOM 元素并添加到页面
   - 创建面板切换按钮（固定在顶栏右侧）
   - 创建面板容器（默认隐藏）
   - 在面板切换按钮的 click 事件中切换面板的显示/隐藏

2. `renderRightPanel()` — 渲染整个右侧面板
   - 清空面板内容
   - 调用三个子渲染函数
   - 将面板插入到 `#app` 的末尾（`document.getElementById('app').appendChild(...)`）

3. `renderRightAgentList(agents)` — 渲染 Agent 状态列表
   - 从 `/api/settings` 获取 agent 数据
   - 每个 agent 显示：小圆点（颜色同侧边栏）+ 名称 + 状态文字

4. `renderRightTokenMeter(data)` — 渲染 Token 仪表
   - 从 `/api/tokens` 获取数据（已由 `refreshTokenStats` 获取）
   - 显示：总 tokens、总花费、缓存命中率
   - 注意：`refreshTokenStats` 已经每 5 秒获取数据，不要重复请求

5. `renderRightTimeline(steps)` — 渲染步骤时间轴
   - 从 `/api/kanban` 获取任务列表
   - 每个步骤显示：状态圆点 + 名称 + 完成标记

**挂载时机（在文件末尾追加）**：

```javascript
// 初始化右侧面板
ready(function() {
  initRightPanel();
  renderRightPanel();
  // 每 10 秒刷新面板数据
  setInterval(renderRightPanel, 10000);
});
```

### 关键约束

- 不在 `index.html` 中添加任何元素。面板的 HTML 完全由 JS 动态创建。
- 面板切换状态保存在 `localStorage` 中（`zero_right_panel`）。
- 桌面端（> 768px）：面板默认显示。
- 移动端（<= 768px）：面板默认隐藏，切换按钮可见。
- 面板宽度 240px。
- 面板切换时，主聊天区的宽度自适应（flex 布局下自动调整）。
