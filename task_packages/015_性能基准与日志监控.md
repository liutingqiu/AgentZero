## 需求

### 需求 1：TokenTracker 增加监控数据采集方法

在 `cognition/token_tracker.py` 的 `TokenTracker` 类中新增方法（如果是模块级函数则新增函数）：

```python
def get_monitor_metrics(self) -> dict:
    """获取聚合监控指标"""
    import time
    now = time.time()

    # 从调用记录中计算指标
    recent_calls = getattr(self, '_recent_calls', []) or []

    # 最近 1 小时内的调用
    hour_ago = now - 3600
    recent = [c for c in recent_calls if getattr(c, 'timestamp', now) > hour_ago]

    # 计算指标
    total_calls = len(recent)
    if total_calls == 0:
        return {
            'total_calls': 0,
            'avg_latency_ms': 0,
            'error_rate': 0,
            'cache_hit_rate': 0,
            'total_tokens': 0,
            'total_cost': 0,
            'period_hours': 1
        }

    latencies = [getattr(c, 'latency', 0) for c in recent]
    errors = [c for c in recent if getattr(c, 'is_error', False)]
    cached = [c for c in recent if getattr(c, 'from_cache', False)]
    tokens = sum([getattr(c, 'tokens_used', 0) for c in recent])
    cost = sum([getattr(c, 'cost', 0) for c in recent])

    return {
        'total_calls': total_calls,
        'avg_latency_ms': round(sum(latencies) / len(latencies) * 1000, 2) if latencies else 0,
        'error_rate': round(len(errors) / total_calls, 4) if total_calls > 0 else 0,
        'cache_hit_rate': round(len(cached) / total_calls, 4) if total_calls > 0 else 0,
        'total_tokens': tokens,
        'total_cost': round(cost, 6),
        'period_hours': 1
    }
```

注意：如果 `TokenTracker` 类中已有类似方法，则不要重复添加，直接使用现有方法。

### 需求 2：新增 `/api/monitor/metrics` 端点

在 `server.py` 中新增路由：

```python
# 路由注册（make_app 函数中）
app.router.add_get('/api/monitor/metrics', handle_monitor_metrics)

# handler
async def handle_monitor_metrics(request):
    """获取监控指标"""
    try:
        from cognition.token_tracker import token_tracker

        # 获取 token tracker 的监控指标
        metrics = {}
        if hasattr(token_tracker, 'get_monitor_metrics'):
            metrics = token_tracker.get_monitor_metrics()
        elif hasattr(token_tracker, 'get_stats'):
            stats = token_tracker.get_stats()
            metrics = {
                'total_tokens': stats.get('total_tokens', 0),
                'total_cost': stats.get('total_cost', 0),
                'cache_hit_rate': stats.get('cache_hit_rate', 0),
            }

        # 添加系统级别指标
        import psutil
        import os

        process = psutil.Process(os.getpid())
        metrics['memory_mb'] = round(process.memory_info().rss / 1024 / 1024, 2)
        metrics['cpu_percent'] = process.cpu_percent(interval=0.1)
        metrics['uptime_seconds'] = int(time.time() - process.create_time())

        return web.json_response({'ok': True, 'metrics': metrics})
    except ImportError:
        # psutil 不可用时返回部分指标
        metrics['memory_mb'] = 0
        metrics['cpu_percent'] = 0
        metrics['uptime_seconds'] = 0
        return web.json_response({'ok': True, 'metrics': metrics})
    except Exception as e:
        return _error_response(ERR_INTERNAL, str(e), status=500)
```

### 需求 3：为每个请求添加 request_id

在 `server.py` 的中间件或路由处理中增加请求追踪 ID。

**方案**：修改 `@middleware` 中间件函数（通常是 `error_middleware` 或 `cors_middleware` 之前的某个中间件）。

```python
@web.middleware
async def request_id_middleware(request, handler):
    """为每个请求添加唯一 request_id"""
    import uuid
    request_id = str(uuid.uuid4())[:8]
    request['request_id'] = request_id

    # 记录请求开始
    logger.info('[%s] %s %s', request_id, request.method, request.path)

    try:
        response = await handler(request)
        # 可选：在响应头中添加 request_id
        response.headers['X-Request-Id'] = request_id
        return response
    except Exception as e:
        logger.error('[%s] %s %s failed: %s', request_id, request.method, request.path, str(e))
        raise
```

**注册位置**：在 `make_app()` 函数中，在 `app.middlewares.append(...)` 的列表中添加：

```python
app.middlewares.append(request_id_middleware)
```

注意：如果 `make_app()` 中已经有中间件列表，按顺序追加。建议放在最外层中间件之后（让它包裹所有其他中间件，确保所有请求都有 ID）。
