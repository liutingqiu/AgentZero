## 需求

### 需求 1：呼吸光圈动画（替换旧思考气泡）

#### 设计要求

```
旧：思考气泡（折叠面板 + "正在思考..." 文本）
    ↓
新：呼吸光圈动画

视觉描述：
- 中心：一个亮度低的白色圆形（#ffffff, opacity 0.3）
- 向外扩散三圈：浅灰 → 浅灰 → 灰
- 三圈从中心向外周期性扩张（类似心跳/呼吸）
- 位于消息区的居中位置，不占消息气泡的宽度
- 思考结束时光圈淡出，消息内容淡入
```

#### JS 部分（app.js）

**1. 替换 `createStreamingRow()`**

旧的 `createStreamingRow()` 创建了一个带思考折叠面板的消息气泡行。改为创建呼吸光圈占位 + 消息容器：

```javascript
function createStreamingRow() {
  const msgs = document.getElementById('msgs');
  const existing = document.getElementById('streamingRow');
  if (existing) return existing;

  const div = document.createElement('div');
  div.className = 'msg-row assistant';
  div.id = 'streamingRow';
  div.innerHTML = ''
    + '<div class="breathing-container" id="breathingContainer">'
    + '<div class="breathing-ring">'
    + '<div class="ring ring-core"></div>'
    + '<div class="ring ring-1"></div>'
    + '<div class="ring ring-2"></div>'
    + '<div class="ring ring-3"></div>'
    + '</div>'
    + '</div>'
    + '<div class="msg-bubble" id="streamingBubble" style="display:none;">'
    + '<div class="msg-content" id="streamingContent"></div>'
    + '</div>'
    + '</div>';
  msgs.appendChild(div);
  msgs.scrollTop = msgs.scrollHeight;
  return div;
}
```

**2. 修改 `updateStreamingContent()`**

当收到第一个 chunk 时，淡出呼吸光圈，显示消息气泡：

```javascript
function updateStreamingContent(text) {
  const el = document.getElementById('streamingContent');
  if (!el) return;

  // 首次收到内容时：淡出光圈，显示气泡
  const container = document.getElementById('breathingContainer');
  const bubble = document.getElementById('streamingBubble');
  if (container && container.style.opacity !== '0') {
    container.classList.add('fade-out');
    if (bubble) {
      bubble.style.display = 'block';
      bubble.classList.add('fade-in');
    }
    // 动画结束后移除光圈 DOM
    setTimeout(function() {
      if (container) container.remove();
    }, 400);
  }

  el.innerHTML = renderMarkdown(text);
  const msgs = document.getElementById('msgs');
  msgs.scrollTop = msgs.scrollHeight;
}
```

**3. 修复流式 chunk 渲染**

如果流式输出没有逐块显示（而是一次性全部显示），问题可能在 SSE 的 `processLine` 中 `type: 'chunk'` 的处理。

在 `send()` 函数的 `processLine` 中，确认 `type === 'chunk'` 时调用 `updateStreamingContent`：

```javascript
if (d.type === 'chunk') {
  accumulatedReply += d.data;
  updateStreamingContent(accumulatedReply);
  // 不需要再切换思考状态——呼吸光圈自动由 updateStreamingContent 处理
}
```

如果 `type` 不是 `chunk` 而是其他字段名（如 `data`、`content`），需要查看 `handle_message` 实际返回了什么。可以在 `processLine` 中添加调试日志：

```javascript
if (d.type === 'chunk' || d.data !== undefined) {
  const content = d.data || d.content || '';
  if (content) {
    accumulatedReply += content;
    updateStreamingContent(accumulatedReply);
    return;
  }
}
```

**4. 修改 `finishStreaming()`**

确保流结束时，如果呼吸光圈还在（没有收到任何 chunk），强制关闭它：

```javascript
function finishStreaming(replyText, conv) {
  // 如果光圈还在（零没有任何输出），强制关闭
  const container = document.getElementById('breathingContainer');
  if (container) {
    container.classList.add('fade-out');
    setTimeout(function() { if (container) container.remove(); }, 400);
  }

  conv.messages.push({ role: 'assistant', content: replyText });
  saveConversations();
  renderMessages();
}
```

## 验收标准

| # | 验收项 | 操作 |
|:-:|-------|------|
| 1 | 呼吸光圈显示 | 发送消息 → 消息区居中显示白色呼吸光圈（三圈渐变向外扩张） |
| 2 | 光圈淡出 | 收到第一个文字 chunk → 光圈淡出消失 |
| 3 | 消息淡入 | 光圈消失后 → 消息内容淡入显示 |
| 4 | 流式逐块 | 文字逐段出现，不是一次性全部显示 |
| 5 | 无旧气泡残留 | 页面上没有旧的思考折叠面板（`thinking-fold`）残留 |
| 6 | 异常处理 | 如果零没有返回任何内容，光圈在流结束后自动消失 |
